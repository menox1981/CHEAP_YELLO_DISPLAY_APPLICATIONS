Flash it. Don't fight your toolchain.

A growing set of applications for the ESP32 Cheap Yellow Display, installable straight from this page over USB — no PlatformIO, no Arduino IDE, no drivers beyond a Chromium browser. Pick an application below, click Install, done.

Browser requirement: works in Chrome, Edge, or Opera on desktop only (Windows/macOS/Linux/ChromeOS) — it runs on the Web Serial API, which Firefox, Safari, and mobile browsers don't support. Plug the device in over USB first, then click Install and choose its serial port when prompted.

Rolling Clock

Modular clock & weather firmware. Three builds, same core: pick the one that matches your board and how much you're willing to pay for it.

Premium Version

The original release: multi-provider weather with on-device screens and branding.

  • Open-Meteo + OpenWeatherMap weather, with automatic failover
  • 3 on-device weather screens: Now / Forecast / Stats
  • Custom brand name (boot screen, dashboard, web dashboard)
  • Web dashboard with live diagnostics (/api/health)
  • WiFi setup portal + Improv serial provisioning

Everything in Premium Version, plus an access-code lock, an on-device keyboard, and AI insights.

Premium v3 (Non-Touch)

Same feature set as v2, built for CYD boards with no touch controller.

  • All Premium v2 features — configured without any touch input
  • Access code entered over USB serial (UNLOCK <code>)
  • City, brand, provider, and API keys set via web dashboard / WiFi portal
  • Clock and weather dashboard still render and update normally
  • Ideal for a display-only "digital signage" style install
