Rolling Clock — Web Flasher

Flash a Rolling Clock firmware straight from this page over USB — no PlatformIO, no Arduino IDE, nothing to install. Pick the variant that matches your hardware and click Install.

Browser requirement: this only works in Chrome, Edge, or Opera on desktop (Windows/macOS/Linux/ChromeOS) — it uses the Web Serial API, which Firefox and Safari don't support, and which isn't available on mobile browsers. Plug your ESP32 Cheap Yellow Display in via USB before clicking Install, and pick its serial port when prompted.
Touchscreen required

Premium Version

The original release: multi-provider weather with on-device screens and branding.

  • Open-Meteo + OpenWeatherMap weather, with automatic failover
  • 3 on-device weather screens: Now / Forecast / Stats
  • Custom brand name (boot screen, dashboard, web dashboard)
  • Web dashboard with live diagnostics (/api/health)
  • WiFi setup portal + Improv serial provisioning

Premium v3 (Non-Touch)

Same feature set as v2, built for CYD boards with no touch controller.

  • All Premium v2 features — configured without any touch input
  • Access code entered over USB serial (UNLOCK <code>)
  • City, brand, provider, and API keys set via web dashboard / WiFi portal
  • Clock and weather dashboard still render and update normally
  • Ideal for a display-only "digital signage" style install
Your browser doesn't support Web Serial. Use Chrome or Edge on desktop. Web Serial needs HTTPS. You're seeing this because the page isn't served securely.

After installing, connect the device to WiFi (via the on-screen/serial setup or the CYD-Setup-XXXXXX access point) and open its web dashboard to configure weather, branding, and (on v2/v3) the OpenWeatherMap/OpenAI API keys.

Having trouble flashing? Make sure no other program (Arduino IDE, PlatformIO monitor, etc.) has the serial port open, and try a different USB cable/port if the device isn't detected.
